Smart Tailoring API

Node.js + Express backend replacing the legacy PHP implementation.

Quickstart

  1. Copy .env.example to .env and set MySQL + JWT values.
  2. Run npm install then npm run migrate.
  3. Start the API: npm start.

Health check: GET /api/health

Key Routes

  • POST /api/auth/register/customer
  • POST /api/auth/register/tailor
  • POST /api/auth/login (role required)
  • GET /api/measurements/fields
  • POST /api/orders (customer)
  • PATCH /api/orders/:orderId/status (tailor/admin)

See the repository documentation for full details.
